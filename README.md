## 📂 Featured Projects  

### 🏦 [Andes Bank – Early Churn Warning System](https://github.com/DevEnriquegd/early_churn_warning_system)  
* **Problema:** Tasa de abandono del **20.4%** sin visibilidad de causas raíz.  
* **Solución:** Análisis diagnóstico de churn con enfoque en CX y correlación de quejas.  
* **Impacto:** Identificación de variables críticas con **99.5% de correlación** con el abandono.

### 📡 [Telecom X – Churn Prediction Pipeline (ML)](https://github.com/DevEnriquegd/ca-telecom-x)  
* **Problema:** Deserción del **27%** en clientes de alto valor (Fibra Óptica).  
* **Solución:** Pipeline de ML (Random Forest + Undersampling) optimizado para maximizar **Recall**.  
* **Impacto:** **79% de efectividad** en detección de fuga, permitiendo acciones preventivas automáticas.  

### 🏀 [NBA Player Stats – Data MVP Framework](https://github.com/DevEnriquegd/mvp-nba)  
* **Problema:** Sesgos subjetivos en la evaluación del desempeño histórico de jugadores.  
* **Solución:** Normalización (Min-Max) y ponderación de estadísticas multidimensionales (1996–2023).  
* **Impacto:** Framework objetivo para la comparación de desempeño entre diferentes eras.
